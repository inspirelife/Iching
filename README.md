Iching
======

IChing, BaZi, FengShui, Divine, Lunar Calendar

To build locally:
  1. install node.js
  2. install yeoman
  3. install grunt
  4. install compass
  5. use grunt server to build app locally
  6. use grunt deploy to deploy to heroku

a live demo is on heroku:
  http://ichingapp.herokuapp.com/
